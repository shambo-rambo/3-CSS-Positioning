CSS Positioning

Static Position

This is the default position type - you don't need to set any position value.

Elements appear in order, one after another, as they are written in the HTML.

Relative Position

RELATIVE means moving something from where it started.
Using position: relative lets us move an element from its normal position, while keeping its original space.
In this example, we used 'left: 200px' to move the box 200 pixels to the right.

Absolute Position

ABSOLUTE means placing something based on its closest positioned container.

Position: absolute lets us place an element exactly where we want it inside its container.

In our code, we used 'top: 50px' to move it down from the top and 'right: 100px' to move it in from the right edge.

For this to work, we need to add 'position: relative' to the section that contains our image so that it is 'relative' to the container.

Fixed Position

Position: fixed keeps an element in the same spot on your screen.

We used 'bottom: 0' and 'right: 0' to stick this box to the bottom-right corner.

Try scrolling - the box will stay in place!
